# Awesome ChatGPT UI

**Update (April 2023)**: I am working on a gallery app as the current list has become challenging to navigate and maintain. If you have any feature requests, please submit an issue on this repo.

---

A list of uncurated open source UIs for ChatGPT API, which includes terminal, web/native/smartwatch apps, extensions, editors, bots, AR/VR, etc.

Tags include primary language and framework, hosting method or option (e.g. Docker), integration (e.g. ElevenLabs), differentiating features (e.g. split screen), and license, in this order.

Note: I have organized these projects based on their Github stars as it is a convenient way to estimate their adoption/popularity, although it may not necessarily reflect their quality.

## Terminal

```bash
curl "https://api.openai.com/v1/chat/completions" \
     -H "Authorization: Bearer YOUR_OPENAI_KEY" \
     -H "Content-Type: application/json" \
     -d "{
     \"model\": \"gpt-3.5-turbo\",
     \"messages\": [{\"role\": \"user\", \"content\": \"YOUR PROMPT\"}]
     }"
```
